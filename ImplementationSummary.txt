Implementation summary
1. Environment and data setup
config.py – Paths, 224 image size, 70/15/15 split, batch size, epochs, SAM ρ, Focal Loss γ, IG steps, seed.
requirements.txt – torch, torchvision, numpy, Pillow, scikit-learn, matplotlib, seaborn, PyYAML, kaggle.
README.md – Dataset link, Kaggle download steps, expected layout, env setup, and usage.
2. Data pipeline
transforms.py – Resize 224×224, Gaussian blur, min–max to [0,1], replicate to 3 channels; train-only: horizontal flip, ±10° rotation, brightness (no shearing).
dataset.py – BrainTumorDataset, 70/15/15 stratified split, class names glioma, meningioma, notumor, pituitary; supports Training/ and Testing/ subfolders; build_dataloaders().
verify_data.py – Quick check of loaders and batch shape/range.
3. Models and baseline training
models.py – EfficientNet-B0 and ResNet-50 (ImageNet pretrained), GAP → Dropout(0.5) → Linear(4); build_model().
losses.py – Focal Loss with γ and optional α.
train.py – Baseline (Adam) or SAM training; saves best checkpoint by validation Macro-F1; --model, --use_sam, --rho, --epochs, etc.
4. SAM
sam.py – Two-step SAM: ascent (θ + ε), backward at θ+ε, restore θ, then optimizer.step().
5. Evaluation
metrics.py – Macro-F1, macro precision/recall, accuracy, AUROC (ovr), confusion matrix.
eval.py – Loads checkpoint, runs test set, writes JSON + Markdown to outputs/results/.
6. Explainability
explainability/gradcam.py – Grad-CAM on last conv of EfficientNet/ResNet, heatmap upsampled to input size.
explainability/integrated_gradients.py – Integrated Gradients with black baseline and configurable steps.
explain.py – Grad-CAM and IG on selected test samples, saves 4-panel figures (input, Grad-CAM, IG, overlay) under outputs/results/explainability/.
7. Reproducibility and structure
.gitignore – data/, outputs/, venv/, pycache.
Seeds set in train.py, eval.py, explain.py from config.SEED.
Checkpoints include config and are named best_<model>_<baseline|sam>.pt.
Next steps for you

Create a venv and install deps: pip install -r requirements.txt.
Download the dataset (see README) and run python verify_data.py.
Train baseline: python train.py --model efficientnet_b0 --epochs 30.
Train with SAM: python train.py --model efficientnet_b0 --use_sam --rho 0.05 --epochs 30.
Evaluate: python eval.py --checkpoint outputs/checkpoints/best_efficientnet_b0_baseline.pt.
Explain: python explain.py --checkpoint outputs/checkpoints/best_efficientnet_b0_baseline.pt.
